## ğŸ“– Overview

**CamDroid** is a local-network camera streaming tool that effortlessly bridges your mobile device (or an additional webcam) and your desktop. It eliminates the need for USB cables by using your local Wi-Fi.

The connection is established using a simple, secure **QR Code scan**, after which video frames are securely streamed over HTTP via a Flask backend to a sleek, modern PyQt5 desktop receiver.

- **ğŸš€ Instant QR Pairing**: No typing IP addresses. Just scan the QR code generated by the receiver to instantly connect.
- **ğŸ”’ Secure Token Validation**: Each session generates a unique UUID-based token to ensure only your authorized device can stream to your PC.
- **âš¡ Low-Latency Local Streaming**: Video frames are transmitted directly over your local network using fast `HTTP POST` requests.
- **ğŸ¨ Sleek Desktop GUI**: The receiver boasts a borderless, modern PyQt5 interface with smooth animations and intuitive controls.
- **ğŸ“± Cross-Platform Sender**: The sender is built using Kivy, allowing it to be compiled and run seamlessly on Android devices.
- **ğŸ”„ Camera Switching**: Easily toggle between front and rear cameras on the sender device.

## ğŸ› ï¸ Installation

Both the Sender and Receiver have separate requirements to keep dependencies clean.

### 1. Clone the repository
```bash
git clone https://github.com/stebyvarghese1/Camdroid.git
cd Camdroid
```

### 2. Set up the Receiver (Desktop)
1. Navigate to the project folder (or set up a virtual environment).
2. Install the required dependencies:
```bash
pip install -r requirements_receiver.txt
```
*(Dependencies: `numpy`, `opencv-python`, `qrcode`, `Pillow`, `Flask`, `PyQt5`)*

### 3. Set up the Sender (Mobile / Client)
1. Install the required dependencies for the sender:
```bash
pip install -r requirements_sender.txt
```
*(Dependencies: `kivy`, `opencv-python`, `pyzbar`, `requests`)*

> **Note**: To run the sender natively on an Android phone, you can package `sender.py` into an APK using [Buildozer](https://buildozer.readthedocs.io/en/latest/).

## ğŸš€ Usage

### Step 1: Start the Receiver
On your desktop Windows/Mac/Linux machine, run:
```bash
python receiver.py
```
- A sleek window will pop up.
- Click **"Generate QR"** to display the connection QR code. 

### Step 2: Start the Sender
On the device you want to use as a camera, run:
```bash
python sender.py
```
*(Or launch the compiled Android App).*
- Click **"Scan QR"**.
- Point your device's camera at the QR code on your desktop screen.
- The app will securely connect to the receiver.
- Hit **"Start Streaming"** and enjoy your wireless webcam feed!
